# Raw TCAD Data

Place the Sentaurus Workbench and TCAD curve files in this folder before running the notebook.

Required files:

```text
gtree.dat
IdVgsSat_n<node>_des.plt
```

The notebook expects filenames like:

```text
IdVgsSat_n3360_des.plt
IdVgsSat_n9262_des.plt
```

Generated `.csv` files may also appear here after parsing `.plt` files.
